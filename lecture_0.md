## Course Description

- Introduction to digital signals, number systems, and codes
- Logic gates, Boolean algebra, and digital circuits
- Integrated circuits and logic families (TTL, CMOS)
- Combinational and sequential circuits, timing diagrams
- State diagrams and implementation
- Microprocessor and microcontroller architecture
- Registers, memory, timers/counters, and interrupts
- ALU, I/O applications, A/D conversion, and serial interfaces
- Hardware description language (VHDL) and digital circuit laboratory

## Topics to be Covered

- Boolean Algebra, Boolean Functions, VHDL, and Gates Analysis of circuits
- Number Conversions, Codes, and Function Minimization
- Introduction to Logic Circuit Analysis and Design
- Combinational Logic Circuit Design with VHDL
- Bistable Memory Device Design with VHDL
- Simple Finite State Machine Design with VHDL
- Register transfer level (RTL) design
- Control Unit and Datapath
- Dedicated Microprocessor applications

## Textbook

- Stephen Brown, Zvonko Vranesic, Fundamentals of Digital Logic with VHDL Design 3rd Edition, 2009, McGraw-Hill
